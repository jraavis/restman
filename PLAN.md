# Restman â€” Development Plan

Privacy-first Tauri v2 (Rust + React) REST client. Frontend talks to the backend only over Tauri IPC; Rust owns all networking, file I/O, and crypto. Secrets (tokens, passwords, API keys) never cross IPC in plaintext and are never persisted to SQLite in plaintext â€” see the mask-on-write contract in `src-tauri/src/secrets.rs` and `SECRET_MASK` in `src/lib/types.ts`. Storage is SQLite only.

Repo: https://github.com/jraavis/restman (public, default branch `main`).
